# DropEck-App

DropEck is a macOS menu bar utility for quick access to your clipboard, notes, snippets, tasks, and dropzone files.

## Features

- **Recent** — browse clipboard history and quickly copy items again
- **Workspace** — keep simple notes in one place
- **Snippets** — save useful text, images, and files
- **Tasks** — add and manage quick tasks
- **Dropzone** — keep files and folders ready to access
- **Command Bar** — use slash commands for fast actions
- **Menu Bar Access** — open DropEck directly from the menu bar
- **Custom Terminal Appearance** — light or dark terminal style and custom terminal text color

## Installation

1. Download the latest **DropEck.zip** from the **Releases** page.
2. Unzip it.
3. Move **DropEck.app** to your **Applications** folder.
4. Open the app.

### If macOS blocks the app

Because the app may not yet be notarized, macOS might warn that it is from an unidentified developer.

To open it anyway:

1. Right-click **DropEck.app**
2. Click **Open**
3. Confirm the prompt

You usually only need to do this once.

## Commands

### Navigation
- `/1`
- `/2`
- `/3`
- `/4`
- `/5`
- `/recent` or `/re`
- `/workspace` or `/ws`
- `/snippets` or `/snp`
- `/tasks` or `/tk`
- `/dropzone` or `/dz`

### Clipboard
- `/pin`
- `/unpin`
- `/delete`
- `/pasteplain`

### Snippets
- `/snippet [title]`
- `/tags <tag>`

### Notes and Tasks
- `/note <text>`
- `/task <text>`

### Help
- `/help`

## Shortcuts and Gestures
- Top edge pull — show panel
- Upward pull — hide panel
- Left edge pull — show left panel
- Right edge pull — show right panel
- `Space` on a selected snippet — quick preview

## Requirements

- macOS
